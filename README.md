# ✨ Manga Illusion Studio

Editor de cómics/manga 100% en el navegador — sin cuentas, sin subscripciones, sin servidor. Arma tus páginas con personajes, fondos, efectos y bocadillos de diálogo, y exporta todo a PDF.

Un proyecto de **[Nautilus Lab](https://nautlab.itch.io/)**.

## 🎨 Features

- **Editor visual por drag & drop** — arrastra personajes y efectos directo al canvas
- **Assets propios** — sube tus propias imágenes (PNG/JPG/WEBP) para personajes, fondos y efectos
- **Redimensiona y mueve todo** — cada elemento se puede arrastrar y cambiar de tamaño desde el canvas
- **4 layouts de panel** — 1, 2, 3 o 4 viñetas por página
- **Bocadillos de diálogo** — texto editable con tamaño ajustable
- **Múltiples páginas** — navega entre páginas anteriores/siguientes sin perder nada
- **Asistente de guion** — genera tips e instrucciones automáticas basadas en tu historia
- **Exporta a PDF** — tu cómic listo para compartir o imprimir
- **100% local** — todo corre en tu navegador, tus imágenes nunca salen de tu computadora

## 🚀 Usar

Abre [manga-illusion.html](./index.html) directo en tu navegador — no necesita instalación ni servidor.

1. Escribe tu historia en la pestaña **📝 Historia**
2. Elige un layout en **📐 Layout**
3. Arrastra personajes y efectos al canvas desde **🎨 Assets**, o sube los tuyos
4. Agrega diálogos desde **💬 Texto**
5. Selecciona cualquier elemento para moverlo o cambiarle el tamaño desde la esquina
6. Cuando esté lista tu página, dale **📄 Nueva Página** para seguir la historia
7. Exporta todo con **💾 Exportar PDF**
